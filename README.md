# ☁️ Full-Stack Angular + .NET Cloud Deployment Platform

> Production-grade full-stack application deployed on AWS with automated CI/CD pipelines

---

## 📌 Overview

This project is a **full-stack enterprise-grade application** built using Angular (frontend) and ASP.NET Core (backend), deployed on AWS infrastructure with fully automated CI/CD pipelines.

The system demonstrates **end-to-end development, deployment, and DevOps integration**, focusing on scalability, reliability, and automation.

---
-
## 🏗️ System Architecture

- Angular frontend (component-based UI)
- ASP.NET Core backend (REST APIs)
- AWS EC2 hosting environment
- IIS for backend service management
- CI/CD pipelines using GitHub Actions
